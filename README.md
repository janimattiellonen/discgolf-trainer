# Disc golf trainer

## Introduction

A tool for tracking disc golf skill over time.

### develop (client)

- `cd client`
- `yarn`
- `cp .env.example .env`
- `yarn run start`
  - open browser and go to http://localhost:8888

- `yarn run test`
- `yarn run flow`
- `yarn run lint`

### develop (server)

- `cd server`
- `yarn`
- `cp .env.example .env`
- `yarn run start`

### build (client)

Builds to `dist/` folder.

- `yarn run build`
